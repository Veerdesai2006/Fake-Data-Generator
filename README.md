# Fake-Data-Generator
This project generates realistic fake user data such as names, phone numbers, addresses, and email IDs.

## 🚀 Features
- Generates random full names  
- Creates realistic phone numbers  
- Builds structured addresses  
- Generates email IDs  
- Produces multiple records using loops  

---

## 🧠 Concepts Used
- random module (`choice`, `randint`)
- Lists (data storage)
- Loops (data generation)
- String formatting (f-strings)

---

## 💻 Tech Stack
- Python
